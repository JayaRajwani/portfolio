# Jaya Rajwani — Portfolio

Personal technical portfolio site. Static HTML/CSS/JS — no build step required.

## Stack
- Plain HTML, CSS, vanilla JS
- IBM Plex Mono + Plex Sans (Google Fonts, CDN)
- Static personal site

## Local preview
Open `index.html` directly in a browser, or serve locally:

```bash
# any one of these works
python3 -m http.server 8000
# then visit http://localhost:8000
```

## Maintenance note
This repository is the source for Jaya's personal portfolio. Deployment and domain configuration are intentionally not documented here.

index.html   # single-page site (all sections)
